# LLM Service API with Rust (LLM SDK for Rust)

### Environment Variables

> Generate GPT API KEY : https://platform.openai.com/api-keys
> Generate GEMINI API KEY : https://aistudio.google.com/api-keys

```
LLM_PROVIDER=gpt or gemini
OPENAI_API_KEY={if LLM_PROVIDER is gpt, this field is required}
GOOGLE_API_KEY={if LLM_PROVIDER is gemini, this field is required}
```
